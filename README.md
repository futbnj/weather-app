## How to run an application
1. Get API authentication token from https://openweathermap.org/
2. Run 
``
npm install
``
3. Set the token with 
``
node weather.js -t YOUR_TOKEN
`` 
4. Set the city with 
``
node weather.js -s YOUR_CITY
`` 
5. Run the application with
   ``
   npm run weather
   `` 
OR
``
   node weather.js
   ``
